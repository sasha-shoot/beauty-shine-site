# Як задеплоїти Beauty & Shine сайт

## Що це таке

Це **робочий каркас** сайту-магазину Beauty & Shine на Next.js. Усі сторінки
вже працюють, дизайн на бренді. Каталог поки порожній — наповнимо в Етапі 2.

## Структура

```
beauty-shine-site/
├── app/                # Сторінки (Next.js App Router)
│   ├── page.tsx        # Головна
│   ├── catalog/        # Каталог
│   ├── product/[slug]/ # Картка товару (динамічна)
│   ├── cart/           # Кошик
│   ├── checkout/       # Оформлення
│   ├── confirmation/   # Підтвердження замовлення
│   ├── about/          # Про нас
│   ├── delivery/       # Доставка й оплата
│   ├── contacts/       # Контакти
│   ├── faq/            # Часті питання
│   ├── not-found.tsx   # 404
│   ├── layout.tsx      # Хедер + футер
│   └── globals.css     # Базові стилі + Tailwind
├── components/
│   ├── Header.tsx
│   ├── Footer.tsx
│   └── Logo.tsx
├── tailwind.config.ts  # Бренд (кольори, шрифти, анімації)
├── package.json
└── (інші конфіги)
```

## Деплой на Vercel (рекомендований шлях)

### 1) Створи акаунт GitHub якщо ще нема
https://github.com/signup — реєстрація 1 хв.

### 2) Створи новий приватний репозиторій
- На github.com → New repository
- Назва: `beauty-shine-site`
- Приватний (Private)
- НЕ створюй README, .gitignore — у нас вже є

### 3) Завантаж код через GitHub Desktop
Так само, як з ботом:
- Відкрий GitHub Desktop
- File → Add Local Repository → обери цю папку
- Натисни **Publish repository** → обери `beauty-shine-site` (приватний)
- Готово

### 4) Підключи до Vercel
- https://vercel.com → Sign up with GitHub
- Add New → Project → Import репозиторій `beauty-shine-site`
- Vercel сам визначить що це Next.js — нічого не міняй
- Натисни **Deploy**
- За ~1 хвилину сайт буде онлайн за адресою типу
  `beauty-shine-site.vercel.app`

### 5) Подальші оновлення
Як з ботом: змінив файли → GitHub Desktop commit + push → Vercel автоматично
переробить деплой за 30-60 секунд.

## (Опційно) Локальна розробка

Якщо хочеш бачити зміни одразу під час редагування:

1. Встанови **Node.js** 20+: https://nodejs.org
2. У папці проєкту відкрий термінал
3. `npm install` — встановити залежності (один раз)
4. `npm run dev` — запустити локальний сервер
5. Відкрий http://localhost:3000

Зміни у файлах оновлюються миттєво. Зупинити — Ctrl+C у терміналі.

**Якщо локально не хочеш** — нічого страшного. Просто пушай у GitHub,
Vercel автоматично будує і деплоїть.
