# Beauty & Shine — сайт-магазин

Магазин засобів для догляду за руками й ногами. Бренди divapharm та Veratin.

## Стек

- **Next.js 14** (App Router) + **TypeScript**
- **Tailwind CSS** з власною бренд-палітрою
- **Vercel** — хостинг + автодеплой з GitHub
- **Google Fonts**: Playfair Display (заголовки), Manrope (текст)

## Команди

```bash
npm install     # встановити залежності (один раз)
npm run dev     # локальний сервер http://localhost:3000
npm run build   # збірка для продакшну
npm run lint    # перевірка коду
```
